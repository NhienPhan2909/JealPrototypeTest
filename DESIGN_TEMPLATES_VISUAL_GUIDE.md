## Accessibility Features

- **High Contrast Colors**: Color swatches have borders for visibility
- **Keyboard Navigation**: All interactive elements are keyboard accessible
- **Screen Reader Support**: Buttons and fields have proper labels
- **Color Tooltips**: Hover over color swatches to see hex codes
- **Clear Labels**: All form fields clearly labeled
