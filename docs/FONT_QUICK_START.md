# Quick Start Guide: How to Change Your Website Font

## For Dealership Administrators

### Step-by-Step Instructions

#### 1. Log In to Admin Panel
- Open your browser
- Navigate to: `http://your-website.com/admin/login`
- Enter your admin credentials
- Click "Log In"

#### 2. Go to Dealership Settings
- After logging in, you'll see the admin menu
- Click on **"Dealership Settings"**
- This page lets you customize your dealership's website

#### 3. Find the Website Font Section
- Scroll down until you see **"Website Font"**
- You'll see a dropdown menu with font options
- Below it is a preview box showing how the font looks

#### 4. Choose Your Font
Select from these options:

| Font | When to Use It |
|------|----------------|
| **System Default** | Modern, clean - good for most dealerships |
| **Arial** | Professional, easy to read |
| **Times New Roman** | Traditional, formal - good for luxury dealerships |
| **Georgia** | Elegant, readable - sophisticated feel |
| **Verdana** | Very easy to read - accessibility focused |
| **Courier New** | Technical, retro look |
| **Comic Sans MS** | Casual, friendly - fun dealerships |
| **Trebuchet MS** | Contemporary, modern |
| **Impact** | Bold, eye-catching - for strong branding |
| **Palatino** | Sophisticated, book-like quality |

#### 5. Preview Your Selection
- As soon as you select a font, the preview box updates
- Read the sample text to see how it looks
- Try different fonts until you find one you like

#### 6. Save Your Changes
- Scroll to the bottom of the page
- Click the **"Save Settings"** button
- Wait for the success message: "Dealership settings updated successfully!"

#### 7. View Your Website
- Open a new tab
- Go to your dealership's public website
- **All text on your website now uses the new font!**

### Tips

**Want a professional look?**
- Try **Arial** or **Verdana**

**Want a traditional, upscale look?**
- Try **Times New Roman** or **Georgia**

**Want a modern, tech-savvy look?**
- Stick with **System Default** or try **Trebuchet MS**

**Want easy reading for all customers?**
- Try **Verdana** (designed for screen readability)

### FAQ

**Q: Will this affect other dealerships' websites?**
A: No! Each dealership has its own independent font setting.

**Q: Can I change it back?**
A: Yes! Just select "System Default" and save again.

**Q: Do I need to refresh the website?**
A: The font changes apply immediately. Just refresh the page to see updates.

**Q: Will mobile users see the new font?**
A: Yes! The font applies to all devices - desktop, tablet, and mobile.

**Q: What if I don't like any of these fonts?**
A: Contact support for information about custom font options (future feature).

### Need Help?

If you encounter any issues:
1. Check that you clicked "Save Settings"
2. Look for the green success message
3. Try refreshing your browser (Ctrl+F5 or Cmd+Shift+R)
4. Contact technical support if problems persist
