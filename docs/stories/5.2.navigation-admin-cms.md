# Story 5.2: Navigation Manager Admin CMS UI

**Epic:** 5 - Navigation Button Enhancement
**Story Number:** 5.2
**Created:** 2025-12-01

## Status

## Story

**As a** dealership administrator,
**I want** to customize my website's navigation buttons with icons and text from the admin panel,
**so that** I can tailor the navigation to match my dealership's branding and priorities without developer assistance.

---

## Acceptance Criteria

1. "Navigation Manager" section added to Dealership Settings page (`/admin/settings`)
2. Navigation Manager displays all current navigation items from dealership.navigation_config (or defaults if null)
3. Each navigation item shows: icon preview, label text, route, enabled toggle, drag handle, delete button
4. Icon picker component allows selecting from popular react-icons (FontAwesome, Material Icons)
5. Icon picker has search functionality to filter icons by name
6. Text input fields allow editing navigation item labels (e.g., "Home" → "Welcome")
7. Drag-and-drop interface allows reordering navigation items (updates order field)
8. Enable/disable toggle for each navigation item (updates enabled field)
9. "Add Navigation Item" button creates new item with default values
10. Delete button removes navigation item (with confirmation modal)
11. Live preview panel shows header with current navigation config (updates in real-time)
12. Live preview applies theme color to buttons (uses existing theme system)
13. Save button sends PUT request to `/api/dealers/:id` with updated navigation_config
14. Success message displayed after successful save: "Navigation settings saved successfully!"
15. Error message displayed if save fails (displays API error)
16. Validation prevents saving navigation with invalid data (empty labels, duplicate ids, invalid routes)
17. "Reset to Defaults" button restores default navigation configuration
18. Navigation Manager is mobile-responsive (works on tablet/iPad)
19. Changes do not affect other dealerships (multi-tenant isolation)
20. Existing dealership settings (theme color, font family) remain functional
21. **[ENHANCEMENT 2025-12-03]** "Show Icon" toggle allows hiding icons while keeping text visible for each navigation button

**NavigationManager Component Interface:**
```jsx
import { useState } from 'react';

function NavigationManager({ dealership, onSave }) {
  const [navItems, setNavItems] = useState(
    dealership.navigation_config || defaultNavigation
  );
  const [isSaving, setIsSaving] = useState(false);

  const handleSave = async () => {
    setIsSaving(true);
    try {
      const response = await fetch(`/api/dealers/${dealership.id}`, {
        method: 'PUT',
        headers: { 'Content-Type': 'application/json' },
        credentials: 'include',
        body: JSON.stringify({ navigation_config: navItems })
      });
      if (response.ok) {
        // Success message
        onSave(); // Refresh dealership data
      } else {
        // Error message
      }
    } finally {
      setIsSaving(false);
    }
  };

  // ... drag-and-drop, CRUD operations
}
```

**Drag-and-Drop Integration:**
```jsx
import { DragDropContext, Droppable } from '@hello-pangea/dnd';

function NavigationManager() {
  const handleDragEnd = (result) => {
    if (!result.destination) return;

    const items = Array.from(navItems);
    const [reorderedItem] = items.splice(result.source.index, 1);
    items.splice(result.destination.index, 0, reorderedItem);

    // Update order field for all items
    const updatedItems = items.map((item, index) => ({
      ...item,
      order: index + 1
    }));

    setNavItems(updatedItems);
  };

  return (
    <DragDropContext onDragEnd={handleDragEnd}>
      <Droppable droppableId="navigation-items">
        {(provided) => (
          <div {...provided.droppableProps} ref={provided.innerRef}>
            {navItems.map((item, index) => (
              <DraggableNavItem
                key={item.id}
                item={item}
                index={index}
                onUpdate={handleUpdate}
                onDelete={handleDelete}
              />
            ))}
            {provided.placeholder}
          </div>
        )}
      </Droppable>
    </DragDropContext>
  );
}
```

**Navigation Item Data Model:**
```javascript
{
  id: string,           // Unique identifier (e.g., 'home', 'nav-1234567890')
  label: string,        // Button text label (e.g., 'Home', 'Inventory')
  route: string,        // Route path (e.g., '/', '/inventory')
  icon: string,         // Icon name from react-icons (e.g., 'FaHome', 'FaCar')
  order: number,        // Display order (1-based, determines position)
  enabled: boolean,     // Whether button appears in navigation
  showIcon: boolean     // [ADDED 2025-12-03] Whether icon is visible (text always shows)
}
```

**showIcon Field Details:**
- **Type:** Boolean (optional, defaults to `true` if undefined)
- **Purpose:** Controls icon visibility independently from text
- **Behavior:** When `false`, only button text displays; when `true`, both icon and text display
- **Backwards Compatibility:** Existing navigation configs without `showIcon` default to showing icons
- **UI Control:** Toggle switch labeled "Show Icon" in DraggableNavItem component
