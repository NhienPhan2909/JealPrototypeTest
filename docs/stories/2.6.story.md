# Story 2.6: About/Contact Page

## Status

**Done**

## Story

**As a** car buyer,
**I want** to view dealership information including location, hours, and contact details,
**so that** I can visit the dealership or contact them outside of vehicle-specific enquiries.

## Acceptance Criteria

1. About/Contact page route `/about` created and accessible from navigation
2. Page fetches dealership information from API (`GET /api/dealers/:id`)
3. Page displays dealership name as H1 heading
4. "About Us" section displays dealership `about` text (2-3 paragraphs)
5. Contact information section displays: Address (formatted), Phone (clickable `tel:` link on mobile), Email (clickable `mailto:` link)
6. Hours section displays dealership `hours` (formatted as table or list)
7. Logo displayed prominently if `logo_url` exists
8. Optional: embed Google Maps iframe or static map image showing dealership address
9. "Browse Our Inventory" call-to-action button links to `/inventory`
10. Page is mobile-responsive with readable text and properly formatted contact info

### API Specification - GET /api/dealers/:id

**Endpoint:** `GET /api/dealers/:id`

**Authentication:** None required (public endpoint)

**URL Parameter:**
- `id` (integer, required): Dealership ID (hardcoded to 1 for MVP)

**Success Response (200):**
```json
{
  "id": 1,
  "name": "Acme Auto Sales",
  "logo_url": "https://res.cloudinary.com/...",
  "address": "123 Main St, Springfield, IL 62701",
  "phone": "(555) 123-4567",
  "email": "sales@acmeauto.com",
  "hours": "Mon-Fri 9am-6pm, Sat 10am-4pm, Sun Closed",
  "about": "Family-owned dealership serving Springfield for 20 years.",
  "created_at": "2025-11-19T10:00:00.000Z"
}
```

**Error Response (404):**
```json
{
  "error": "Dealership not found"
}
```

**Usage in About.jsx:**
```javascript
const response = await fetch(`/api/dealers/1`);
if (response.ok) {
  const data = await response.json();
  setDealership(data);
} else {
  setError('Unable to load dealership information. Please try again later.');
}
```

### Data Model - Dealership

**Dealership Object Structure (from API response):**

```javascript
{
  id: number,                           // Dealership ID (always 1 for MVP)
  name: string,                         // Dealership business name (e.g., "Acme Auto Sales") - NOT NULL
  logo_url: string | null,              // Cloudinary URL for logo image (nullable)
  address: string,                      // Full street address for contact page - NOT NULL
  phone: string,                        // Contact phone number (e.g., "(555) 123-4567") - NOT NULL
  email: string,                        // Contact email address - NOT NULL
  hours: string | null,                 // Business hours (e.g., "Mon-Fri 9am-6pm...") - nullable
  about: string | null,                 // About us description for public website - nullable
  created_at: string                    // ISO timestamp (e.g., "2025-11-19T10:00:00.000Z")
}
```

**Field Handling:**
- **Required fields:** name, address, phone, email (always present in API response)
- **Optional fields:** logo_url, hours, about (may be null)
- **Nullable field handling:**
  - If `logo_url` is null, don't display logo (no placeholder needed)
  - If `hours` is null, display: "Hours not available. Please call for hours."
  - If `about` is null, display: "No information available."

### Component Pattern - About.jsx

**About Component Structure:**

```javascript
import { useState, useEffect } from 'react';
import { Link } from 'react-router-dom';

/**
 * About - Displays dealership information including contact details, hours, and about text.
 *
 * @component
 * @example
 * <About />
 */
function About() {
  // State
  const [dealership, setDealership] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  // Fetch dealership data on mount
  useEffect(() => {
    const fetchDealership = async () => {
      try {
        const response = await fetch('/api/dealers/1'); // Hardcoded ID for MVP
        if (response.ok) {
          const data = await response.json();
          setDealership(data);
        } else {
          setError('Unable to load dealership information. Please try again later.');
        }
      } catch (err) {
        console.error('Failed to load dealership:', err);
        setError('Unable to load dealership information. Please try again later.');
      } finally {
        setLoading(false);
      }
    };

    fetchDealership();
  }, []);

  // Loading state
  if (loading) {
    return <div className="text-center text-gray-600 mt-10">Loading dealership information...</div>;
  }

  // Error state
  if (error) {
    return (
      <div className="bg-red-50 border border-red-500 text-red-800 p-4 rounded max-w-2xl mx-auto mt-10">
        {error}
      </div>
    );
  }

  // Main content
  return (
    <div className="max-w-4xl mx-auto px-4 py-8">
      {/* Logo */}
      {dealership.logo_url && (
        <img
          src={dealership.logo_url}
          alt={dealership.name + ' Logo'}
          className="mx-auto mb-6 max-w-xs"
        />
      )}

      {/* Dealership Name */}
      <h1 className="text-4xl font-bold mb-6 text-center">{dealership.name}</h1>

      {/* About Us Section */}
      <div className="card mb-6">
        <h2 className="text-2xl font-semibold mb-4">About Us</h2>
        <p className="whitespace-pre-line text-gray-700 leading-relaxed">
          {dealership.about || 'No information available.'}
        </p>
      </div>

      {/* Contact Information Section */}
      <div className="card mb-6">
        <h2 className="text-2xl font-semibold mb-4">Contact Us</h2>
        <dl className="space-y-2">
          <div>
            <dt className="font-semibold">Address:</dt>
            <dd className="text-gray-700">{dealership.address}</dd>
          </div>
          <div>
            <dt className="font-semibold">Phone:</dt>
            <dd>
              <a href={`tel:${dealership.phone}`} className="text-blue-600 hover:underline">
                {dealership.phone}
              </a>
            </dd>
          </div>
          <div>
            <dt className="font-semibold">Email:</dt>
            <dd>
              <a href={`mailto:${dealership.email}`} className="text-blue-600 hover:underline">
                {dealership.email}
              </a>
            </dd>
          </div>
        </dl>
      </div>

      {/* Hours of Operation Section */}
      <div className="card mb-6">
        <h2 className="text-2xl font-semibold mb-4">Hours of Operation</h2>
        <p className="whitespace-pre-line text-gray-700">
          {dealership.hours || 'Hours not available. Please call for hours.'}
        </p>
      </div>

      {/* Call-to-Action Button */}
      <div className="flex justify-center mt-8">
        <Link to="/inventory" className="btn-primary">
          Browse Our Inventory
        </Link>
      </div>
    </div>
  );
}

export default About;
```

### Clickable Links Pattern

**Phone Link (tel:):**
```javascript
<a href={`tel:${dealership.phone}`} className="text-blue-600 hover:underline">
  {dealership.phone}
</a>
```
- On mobile devices, clicking opens phone dialer with number pre-filled
- On desktop, clicking may open default phone app or do nothing (browser-dependent)
- Always use `tel:` protocol for phone numbers

**Email Link (mailto:):**
```javascript
<a href={`mailto:${dealership.email}`} className="text-blue-600 hover:underline">
  {dealership.email}
</a>
```
- Clicking opens default email client with "To:" field pre-filled
- Works on both desktop and mobile
- Always use `mailto:` protocol for email addresses
